TUNING PARAMETER KALMAN FILTER

LIVE TUNING

Process & Measurement Noise

Q

Process Noise

Mengontrol seberapa besar sistem mempercayai prediksi internal vs. pengukuran. Nilai kecil = trust prediksi (hasil halus, lambat reaksi). Nilai besar = trust pengukuran (responsif, lebih noisy).

Range: 0.001 – 1 • Recommended: 0.05

Rᵤ

Measurement Noise — Ultrasonic (HC-SR04)

Variansi pengukuran sensor ultrasonik. HC-SR04 lebih akurat untuk jarak menengah, jadi nilai R relatif kecil. Nilai kecil = sensor dipercaya, hasil mengikuti pembacaan US.

Range: 0.5 – 50 • Recommended: 10.0

Rᵢ

Measurement Noise — Infrared (Sharp GP2Y0A21)

Variansi pengukuran sensor inframerah analog. Sharp GP2Y0A21 unggul di jarak dekat. Sesuaikan nilai R relatif terhadap Rᵤ untuk menentukan bobot sensor mana yang lebih dipercaya.

Range: 0.5 – 50 • Recommended: 5.0

02

REFERENSI ALGORITMA

Sequential 2-Stage Kalman Filter

Algoritma ini melakukan satu prediksi diikuti dua update sensor secara berurutan (HC-SR04 → GP2Y0A21). Output stage pertama menjadi prior untuk stage kedua, sehingga kedua sensor digabungkan dengan bobot otomatis berdasarkan parameter R-nya.

// PREDICT
P_pred = P + Q;

// STAGE 1 — Ultrasonic update
K1     = P_pred / (P_pred + R_ultra);
x_upd1 = x + K1 * (z_ultra - x);
P_upd1 = (1 - K1) * P_pred;

// STAGE 2 — Infrared update (uses stage 1 output as prior)
K2 = P_upd1 / (P_upd1 + R_ir);
x  = x_upd1 + K2 * (z_ir - x_upd1);
P  = (1 - K2) * P_upd1;

DEFAULT Q

0.05

process noise

DEFAULT Rᵤ

10.0

HC-SR04 variance

DEFAULT Rᵢ

5.0

GP2Y0A21 variance
